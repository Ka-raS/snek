# Snek

is snek

## Prerequisites

- [**CMake**](https://cmake.org/download/) (version 3.11 or higher) - building tool
- [**raylib**](https://github.com/raysan5/raylib) (auto fetched by CMake) - library
- **C++17** (or higher)

## Installation

1. Clone:
    ```bash
    git clone https://github.com/Ka-raS/snek.git
    ```

2. Build:
    ```bash
    cmake -B build
    cmake --build build
    ```

3. Run:
    ```bash
    ./build/snek/snek
    ```
